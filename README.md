# Public Health Data Analysis  

This repository contains data-driven research and analysis focused on public health trends and policy implications.

---

## 📊 Project Overview  

This project examines trends and determinants of leading causes of death in the United States using a data-driven approach. The analysis highlights patterns in mortality, explores key contributing factors, and discusses implications for public health policy and planning.

---

## 🔍 Objectives  

- Analyze mortality trends using available datasets  
- Identify key determinants influencing health outcomes  
- Support evidence-based policy discussions  
- Provide insights for public health planning and intervention
